# Women Safety Web Application

## Problem Statement
Women often face unsafe situations where immediate help is required.
This project provides a one-tap SOS system to share live location during emergencies.

## Features
- User Registration & Login (Firebase Authentication)
- SOS Button
- Real-time Location Tracking
- Google Maps Integration
- Firebase Realtime Database

## Technologies Used
- HTML, CSS, JavaScript
- Firebase (Google)
- Google Maps API

## How It Works
1. User registers or logs in
2. Clicks SOS button during emergency
3. Location is captured and shown on map
4. Data is stored securely in Firebase

## Future Scope
- Android Mobile App
- Emergency Contact Alerts
- Live Location Sharing
- AI-based Safety Guidance
